- 以 [Phase 2C PRD](/Users/coderlauu/xRag/docs/prd/2026-04-16-xrag-phase-2c-prd.md) 和 [Phase 2C Product Backlog](/Users/coderlauu/xRag/docs/prd/2026-04-16-xrag-phase-2c-backlog.md) 为准，冻结 `v6 / P0`

- 更新 `current.md / status / exec plan`，保证后续新 session 可恢复

### 明确不做

- 在 `technical-evaluation` 完成前直接进入实现 lane
- 未经新版本技术评估就修改 `schema / shared-types / API contract / citation / scope / eval` 主边界

- 开放互联网联网回答
- 团队协作 / 权限模型
- 浏览器插件
- 移动端
- 全自动 agent 工作流

### 原型 / 文档中存在但本轮只做预留

- `P1-01` Drill-down 与样本回放
- `P1-02` 成本与质量联动分析
- 更深的知识库管理、摘要、推荐、多模型路由能力
- 任何需要重新打开 `Phase 2A / 2B` 已冻结信任边界的方向
